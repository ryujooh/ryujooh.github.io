# Hello World!

Welcome to my new blog. This site is built with **pure HTML, CSS, and Vanilla JavaScript**, and it's powered by **GitHub Pages**.

## Features

- 🚀 **Fast**: Pure static files with no heavy framework.
- 🎨 **Modern Design**: Responsive and minimalist aesthetics.
- 🌓 **Dark Mode**: Supports system preferences and manual toggle.
- 💬 **Comments**: Integrated with Giscus (GitHub Discussions).
- 📝 **Markdown**: Write posts in simple Markdown.

## Code Example

Here is a simple JavaScript snippet:

```javascript
function greet(name) {
  console.log(`Hello, ${name}! Welcome to my blog.`);
}

greet('Visitor');
```
